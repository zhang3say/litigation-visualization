# 诉讼可视化参考手册

## 一、图表类型详解

### 1. 时间图（时间轴线图）

**何时使用**：案件核心争议在于事件发生的时间点、先后顺序或时间区间。

**基本结构**：在纸中间画一条横线（或竖线），标注时间和事件。

**纵向空间利用**：
- **对比关系**：时间轴上方放一方行为，下方放另一方行为，形成对比
- **行为分层**：复杂案件可将时间轴空间分为3-4层，每层放不同类别行为
  - 例：银行行为 / 信用社行为 / 基层法院行为 / 中院行为 / 高院行为

**横向时间轴加工**：
- **括号**：表达「时间区间」，如保证期间、诉讼时效期间
- **箭头**：表达持续的时间区间，如暂缓受理期间
- **线条颜色变化**：区分不同阶段的状态，如不同公章使用时期

**时间图绘制 checklist**：
- [ ] 确定核心时间节点（合同签订日、履行日、到期日等）
- [ ] 确定纵向空间如何分配（对比/分层）
- [ ] 确定是否需要表达「时间区间」（用括号/箭头/颜色变化）
- [ ] 用两种颜色区分不同当事方行为
- [ ] 用红色标注违约/关键行为

### 2. 关系图（法律关系图）

**何时使用**：
- 主体繁多（3个以上且需要梳理关系）
- 关系本身是争议焦点（如融资性贸易、让与担保）
- 关系为叙事逻辑（如股权结构变动）

**关系图绘制要领**：

1. **保持点线一致性**：
   - 主体用相同大小的矩形/圆角矩形
   - 线条含义一致（持股线条全为实线，不能中途变转让线）
   - 同一形状不混用（切忌方圆三角混搭）

2. **主体尽量不重复**：合并同类项，同一主体只出现一次

3. **避免线条交叉、曲折**：通过调整主体排列顺序避免

4. **符合构图习惯**：
   - 股权结构：股东在上，子公司在下
   - 箭头不代表两重含义

**关系图常见结构**：
- **层级图**：股权结构、组织结构（股东上→子公司下）
- **循环图**：融资性贸易闭环（货物/资金流转画成环）
- **放射图**：以核心主体为中心，向四周扩散法律关系

**关系图绘制 checklist**：
- [ ] 确定所有参与主体
- [ ] 选择合适图形结构（层级/循环/放射）
- [ ] 统一主体图形形状和大小
- [ ] 确定线条含义（持股/资金流/合同关系/实际控制）
- [ ] 避免线条交叉
- [ ] 主体位置合理（核心主体在中间，股权结构中股东在上）

### 3. 数据图

**何时使用**：
- 数据量大且繁杂（给付受众带来困难）
- 存在重要数据变动（如多轮增资减资）
- 需要强调数据间对比关系

**表格**：横纵坐标表达不同维度
- 横向：主体、时间
- 纵向：事件、变动情况
- 示例：各股东各时期实际出资 vs 登记出资金额

**统计图表**：
- **柱状图**：数据间对比（如注册资本恢复对比）
- **饼状图**：比例关系（如持股比例）
- **折线图**：趋势变化（如房地产景气指数、价格走势）
- **混合使用**：折线图+事件标注 = 背景趋势+个案事实

### 4. 特殊图表

- **路线图**：地理位置、事故现场、道路走向
- **平面图**：房地产项目布局、施工范围、争议区域标注
- **立体图**：建设工程三维施工情况、山区工程分层
- **流程图**：行政审批程序、票据贴现流程、再审流程

## 二、设计规范

### 色彩使用

**基本配色方法**：
- 同类色（色相环30度）：协调柔和，可能单调
- 邻近色（色相环60度）：色彩融合自然
- 对比色（色相环90度）：丰富且协调，理想关系
- 互补色（色相环180度）：强烈视觉冲击

**诉讼图表配色规则**：
| 颜色 | 含义 | 适用 |
|------|------|------|
| 红色 | 强调、警示、违约 | 对方违约行为、争议焦点、不利事实 |
| 蓝色 | 正常、安全、履约 | 己方履约行为、合规程序 |
| 绿色 | 平静、安全、可行 | 已完成/已结算事项 |
| 灰色 | 模糊、辅助、弱化 | 无证据事实、非核心信息 |
| 黑色 | 中性、客观 | 基本框架、一般事实 |

**配色注意事项**：
- 全案图表保持一致的配色方案
- 有明确目的才使用颜色（不为好看而用）
- 颜色不超过3种为最佳
- 使用低饱和度颜色（法律工作的严肃性）
- 同一主体使用同一颜色
- 填充色与边框色尽量一致

### 构图规范

**对比原则**：通过颜色、字重、填充、大小差异突出重点，差异要足够明显

**重复原则**：同一类型元素保持颜色、线宽、字体、形状统一（如所有「公司」用相同颜色矩形，所有「个人」用另一种）

**对齐原则**：使用参考线，避免元素随意摆放

**亲密性原则**：
- 相关元素放在一起
- 不同领域之间留出空间距离
- 先在结构上给读者分类暗示

**其他技巧**：
- 避免线条交叉
- 避免线条长途奔袭（连接两端尽量靠近）
- 文字合理布局：详细说明放到图表底部/侧面用脚注标注
- 图表不要过于拥挤，留白

## 三、观点表达技巧

### 通过线条表达立场
- 实线 = 实际存在的法律关系/事实关系
- 虚线 = 名义上的/表层的/不存在的关系
- 粗线 = 强调重要性（如大股东权益）
- 箭头 = 资金流向、权利转移方向、控股方向

### 通过颜色表达评价
- 同一颜色用于同一阵营主体
- 用红色标注对方违约，用蓝色标注己方履约
- 用不同颜色区分不同法律关系和资金流向

### 通过框架表达阵营
- 虚线框架 = 利益共同体
- 实线框架 = 合同关系当事人
- 框架的划分方式直接体现了律师对案件的分析结论

### 同一案件不同立场的变化
同一个案件，不同立场的律师会通过改变：
- 框架框定的主体范围
- 实线/虚线的使用
- 颜色的分配
来呈现截然不同的图表。
